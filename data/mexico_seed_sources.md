# Mexico Seed Sources

This file tracks source references used for the Mexico seed dataset.

## Included sources

1. **SAT México**
   - sourceType: `official`
   - sourceUrl: https://www.gob.mx/sat/acciones-y-programas/numeros-telefonicos-falsos
   - note: Official SAT page listing fake/suspicious phone and SMS numbers.

2. **Tellows México**
   - sourceType: `crowd_report`
   - sourceUrl: https://www.tellows.mx/
   - note: Public crowd reports; use medium confidence and manual review.

3. **Seguridad BC - Extorsión Telefónica**
   - sourceType: `official`
   - sourceUrl: https://seguridadbc.gob.mx/ExtorsionTelefonica/index.php
   - note: Official state page provides phone lookup and most reported numbers, but do not invent numbers unless visible and verified manually.
