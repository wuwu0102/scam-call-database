Política de Privacidad – Alerta Número MX

Última actualización: 2026

Alerta Número MX no recopila, almacena ni comparte datos personales de los usuarios.

La aplicación funciona utilizando una base de datos remota de números telefónicos reportados como sospechosos o no deseados. Esta información se descarga al dispositivo del usuario únicamente con el propósito de identificar llamadas entrantes.

Datos personales

La aplicación:

  • No recopila números de teléfono del usuario.
  • No accede al historial de llamadas del usuario.
  • No sube información del dispositivo a ningún servidor.
  • No requiere registro ni inicio de sesión.
  • No comparte datos con terceros.

Funcionamiento local

El procesamiento de datos ocurre localmente en el dispositivo del usuario. La base de datos descargada se utiliza únicamente para habilitar la identificación de llamadas mediante la función de identificación de llamadas del sistema iOS.

Base de datos

Alerta Número MX puede actualizar su base de datos de forma periódica para mejorar la precisión de la identificación de llamadas. La información utilizada por la aplicación se limita a una base de datos pública o generada externamente.

Limitación

Alerta Número MX proporciona información de referencia para ayudar a identificar llamadas potencialmente sospechosas o no deseadas. La aplicación no garantiza que todos los números sospechosos sean identificados ni que todos los números identificados sean necesariamente fraudulentos.

Contacto

Si tienes preguntas sobre esta política, puedes contactar al desarrollador.
